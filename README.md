# PrimeTrade.ai Assignment

This project analyzes the relationship between cryptocurrency market sentiment and trader performance using Hyperliquid trading data and the Fear & Greed Index dataset.

## Objective
The goal of this project is to study how market sentiment affects trader profitability, win rates, trading behavior, and market participation across different cryptocurrency assets.

## Key Analyses
- Average profitability analysis
- Win rate analysis
- Buy/Sell behavioral analysis
- Correlation analysis
- PnL distribution analysis
- Top traded coins analysis
- Top traders analysis
- Coin-wise profitability analysis

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Files Included
- Google Colab Notebook (.ipynb)
- PDF Report
- Dataset files
